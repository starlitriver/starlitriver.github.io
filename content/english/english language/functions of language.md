these are just some things that language is used for

### referential function

the referential function is the sharing of information with an intended audience, presenting the information as factual and objective, commonly using declarative sentences.

### emotive function

also called the expressive function, it communicates emotion

### conative function

the conative function has directions, questions and commands. this aims to cause the audience to react in some way.

### poetic function

the poetic function communicates an idea that isn't directed at the audience.

### phatic function

the phatic function intends to create and maintain social connections between the addresser and the addressee. they are basically there because of the natural flow of language that comes with social conventions, like saying how are you.

### metalinguistic function

this is when you need to talk about languages, as in trying to understand the exact meaning of a word or asking for clarification about the sentences
