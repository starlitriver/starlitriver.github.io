Open class can often receive new members.

The open class is also known as content words, cause they talk about things.


The closed class words are function words, for grammar.

## open class

- noun
- adjective
- verb
- adverb

## closed class

- pronoun
- auxiliary verb
- preposition
- determiner
- interjection
