## Given $E(X^2)=\dfrac{5}{2}$, find the variance.

| x       | -1  | 0   | 1   | 2    | 3   |
| ------- | --- | --- | --- | ---- | --- |
| Pr(X=x) | m   | 2m  | n   | 0.5n | n+m |
| x^2     | 1   | 0   | 1   | 4    | 9   |
|         |     |     |     |      |     |

$4m+2.5n=1$
$\dfrac{5}{2}=m+n+2n+9n+9m$
$\dfrac{5}{2}=10m+12n$
$\dfrac{5}{2}=10m+6.25n$


$E(X)=(\dfrac{-1}{4}+\dfrac{3}{4})^2=\dfrac{1}{4}$
$\text{Var}(X)=\dfrac{5}{2}-\dfrac{1}{4}=\dfrac{9}{4}$
