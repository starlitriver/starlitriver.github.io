in mathematics, the logarithm is the inverse function to [[exponentiation]]

the log function is quite a peculiar thing
its quite understandable if you always remember that it outputs a exponent, that's it.

for $\log_a(b)$, the base $a$ is restricted to be $0<a<1\lor a>1$ 
the input to the log function, $b$, is restricted to be $b>0$
if the restrictions are not met, a math error happens



when you aren't given a base, it is assumed that the base is 10.
that is, $\log(a)=\log_{10}(a)$


simply explained, if $a^b=c$ then $\log_a(c)=b$

for example, you have $2^4=16$ and $\sqrt[4]{16}=2$ and $\log_2(16)=4$
so, the log base 2 of 16 is 4

for example, you have $3^3=27$ and $\sqrt[3]{27}=3$ and $\log_3(27)=3$
so, the log base 3 of 27 is 3

for function $f(x)=\log(x)$, the domain is $(0,\infty)$ and the range is $\mathbb{R}$
this means that the things inside a log function have to be greater than zero


logarithm identities

$a^{\log_a(b)}=b$

$\log_a(1)=0$

$\log_a(b)=\dfrac{\log(a)}{\log(b)}$

$\log_{a}(b)=\dfrac{\log_c(a)}{\log_c(b)}$

$\log_a(b^c)=c\cdot \log_a(b)$

$\log_a(b)=\dfrac{\log_{c}(b)}{\log_{c}(a)}$

$\log_a(b)=\dfrac{1}{\log_b(a)}$

$\log_a(b+c)=\log_a(b)\cdot\log_a(c)$

$\log_a(b-c)=\dfrac{\log_a(b)}{\log_a(c)}$

$\log_a(b\cdot c)=\log_a(b)^c$

$\log_a(b)=\dfrac{\log(b)}{\log(a)}$
